Let's get started!

We've all heard these things:

  • JavaScript has all the cool toys!
  • CFML is old, dying and only for legacy stuff!

Incorrect!

  • CommandBox is here to save the day!
  • All of the Node.js fundamentals are available to CFML developers

Node.js

  • App you install
  • On laptop or server
  • "JavaScript can run anywhere"
  • In web browser (obviously)
  • As a server side language (ala CFML)
  • As a native CLI tool on your laptop

Installing Node.js

  • Go to Nodejs.org
  • Click one of these:

Node.js as a CLI

  • Command Line tools for development
  • Front-end development as advanced greatly because of this
  • All modern JavaScript frameworks have a CLI
  • They're all made with Node
  • Meaning, they're all written in JavaScript

Node.js as a CLI

  • Command line tools are no longer only done in low-level languages like C++

Okay so what's the catch?

There isn't one

  • CommandBox supports all CFML engines
  • Lucee and Railo
  • ACF as far back as version 9
